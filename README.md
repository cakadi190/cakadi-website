# Cak Adi Official Website
Ini adalah repositori untuk website Cak Adi pada domain https://cakadi.id yang bisa kamu unduh dan oprek untuk kebutuhan showoff portofolio kamu ke ranah publik. Oh iya, jangan lupa ya buat ngasih donasi ke saya supaya website ini bisa kamu miliki secara gratis dan update terbaru dari saya dengan berbagai tampilan yang ciamik soro.

Untuk saat ini sedang saya kembangkan versi 2 karena yang lama di https://v1.cakadi.id kelihatannya ribet dan deprecated menggunakan tailwind dan saya putuskan untuk kembali ke bootstrap dan nuxtjs versi 3.9.0.
