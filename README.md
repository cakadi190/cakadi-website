# Portfolio Cak Adi

Portfolio Cak Adi adalah proyek pengembangan website pribadi yang dirancang untuk menampilkan informasi tentang karya, pengalaman, dan keahlian Adi. Website ini bertujuan untuk menjadi platform online yang profesional dan modern untuk memperkenalkan Cak Adi kepada dunia.

## Fitur Utama

- **Halaman Beranda**: Tampilan utama yang menarik dengan informasi singkat tentang Cak Adi.
- **Tentang Saya**: Bagian yang menjelaskan latar belakang, pengalaman, dan keahlian Cak Adi.
- **Portofolio**: Galeri proyek-proyek yang telah dikerjakan oleh Cak Adi.
- **Kontak**: Formulir dan informasi kontak untuk mempermudah komunikasi.

## Teknologi yang Digunakan

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: AdonisJS
- **Framework/Libraries**: React, Bootstrap, jQuery
- **Hosting**: Hosting Biasa

## Cara Menjalankan Proyek

1. Clone repositori ini:
   ```bash
   git clone https://github.com/cakadi190/cakadi-website.git
   ```
2. Buka folder proyek:
   ```bash
   cd cakadi-website
   ```
3. Jika ada dependency, install menggunakan package manager (misalnya npm atau yarn).
   ```bash
   npm install
   ```
4. Jalankan proyek secara lokal:
   ```bash
   npm start
   ```
5. Buka di browser:
   ```
   http://localhost:3000
   ```
